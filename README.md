# 関数
```javascript
newBlock(x, y, G, H, I, J)
//x,y : 新しいブロックを配置する座標
//G,H,I,J : コネクタを付ける方向、それぞれの項目にtrue,falseで指定
addBlock(direction,  G, H, I, J)
//direction : ブロックを追加する方向 "up","down","left","right"で指定
//G,H,I,J : 上記同様
```
